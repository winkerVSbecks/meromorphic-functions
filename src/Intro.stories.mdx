# Meromorphic Functions

A tool for generating static or animated mesh gradients using a [meromorphic function](https://en.wikipedia.org/wiki/Meromorphic_function).

### Usage

Each visualization is generated using 4 points (x,y) and a color palette. You can chose between a static or animation visualization. Then use the controls to adjust the colour palette and mode.

- The static visualization allows you to control the location of the points.
- The animation visualization allows you to control the animation type.

### Exporting images

- <kbd class="key">spacebar</kbd> to pause or resume the animation

- <kbd>
    <kbd class="key">cmd</kbd> + <kbd class="key">s</kbd>
  </kbd> to export the image

- <kbd>
    <kbd class="key">cmd</kbd> + <kbd class="key">shift</kbd> +{' '}
    <kbd class="key">s</kbd>
  </kbd> to export an image sequence of the animation

### How does it work?

The gradients are generated by visualizing the ratio of two complex polynomials using GLSL and [canvas-sketch](https://github.com/mattdesl/canvas-sketch). The shader is based on Harley Turan's <a href="https://hturan.com/writing/complex-numbers-glsl">Visualizing Complex Numbers Using GLSL</a>.
