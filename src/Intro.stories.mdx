# Meromorphic Functions

A tool for generating static and animated mesh gradients using a [meromorphic function](https://en.wikipedia.org/wiki/Meromorphic_function).

![](demo.png)

### Usage

Each visualization is generated using 4 points (x,y) and a color palette. You can chose between a static or animation visualization. Then use the controls to adjust the parameters:

- Color palette
- Color mode
- Location of the points (for the The static visualization)
- Motion type (for the animated visualization)

#### Exporting images

- <kbd class="key">spacebar</kbd> to pause or resume the animation

- <kbd>
    <kbd class="key">cmd</kbd> + <kbd class="key">s</kbd>
  </kbd> to export the image

- <kbd>
    <kbd class="key">cmd</kbd> + <kbd class="key">shift</kbd> +{' '}
    <kbd class="key">s</kbd>
  </kbd> to export an image sequence of the animation. Then use <a href="https://ffmpeg.org/">
    ffmpeg
  </a> to convert it into a GIF or a video.

### How does it work?

The gradients are generated by visualizing the ratio of two complex polynomials using GLSL and [canvas-sketch](https://github.com/mattdesl/canvas-sketch). The underlying shader is based on Harley Turan's <a href="https://hturan.com/writing/complex-numbers-glsl">Visualizing Complex Numbers Using GLSL</a>.
